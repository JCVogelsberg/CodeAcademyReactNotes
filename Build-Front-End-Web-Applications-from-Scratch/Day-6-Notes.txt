Day 6  - Functions
-------------------------

Functions 

- the building blocks of all JS programs

- Like the "verbs" of programming, when they are executed, they put things into motion

- Used to organize chunks of code that can be reused repeatedly

------------------------- 

Creating Functions
- Function declaration:

  function exampleFunction() {
    console.log("I'm an example function.");
  }

-------------------------

New in ES6: Default parameters

- allow parameters to have a predetermined value in case there is no argument passed into the function or if the argument is undefined when called.

Example:
	function greeting (name = 'stranger') {
	  console.log(`Hello, ${name}!`)
	}

* stranger is the default parameter


Return statement: captrues the result of a function to display
